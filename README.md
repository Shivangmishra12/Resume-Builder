# Resume Builder Fullstack

A simple full-stack resume builder with a static frontend, Express backend, and SQLite persistence.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open `http://localhost:3000`

## Features

- Frontend preview + PDF export
- Save resume data to SQLite
- Load saved resume on refresh
- REST API: `GET /api/resume`, `POST /api/resume`
